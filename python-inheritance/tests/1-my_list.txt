===========================
How to use 1-my_list.py
===========================

Import:
    >>> MyList = __import__('1-my_list').MyList

Check instantiation:
    >>> my_list = MyList()
    >>> type(my_list)
    <class '1-my_list.MyList'>

Check inherits from list:
    >>> issubclass(MyList, list)
    True

Check __str__:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> str(my_list)
    '[1, 2]'

Check append():
    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.append(3)
    >>> print(my_list)
    [5, 3]

Check print_sorted() with sorted append:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [1, 2, 3]

Check print_sorted() with not sorted append:
    >>> my_list = MyList()
    >>> my_list.append(3)
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.print_sorted()
    [1, 2, 3]

Check print_sorted() with negative number:
    >>> my_list = MyList()
    >>> my_list.append(-1)
    >>> my_list.append(-3)
    >>> my_list.append(-2)
    >>> my_list.print_sorted()
    [-3, -2, -1]

Check print_sorted() with empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Check print_sorted() returns a new list (original unchanged):
    >>> my_list = MyList()
    >>> my_list.append(3)
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.print_sorted()
    [1, 2, 3]
    >>> print(my_list)
    [3, 1, 2]
